Grundlæggende web

De vigtigste værktøjer og principper inden for webudvikling og design.

Viden:

  • HTML & CSS: At skrive semantisk markup og opbygge hierarkiske filstrukturer.
  • Responsivt Webdesign: Brug af CSS Grid, Flexbox og media queries med en “mobile-first” tilgang.
  • Designprincipper: Anvendelse af Gestalt-principper, wireframes, style tiles og designkonventioner til at skabe brugervenlige grænseflader.
  • Værktøjer: Brug af Figma til mock-ups og Visual Studio Code med relevante plugins til kodning.

Forståelse:

  • Digitale Formater: Arbejde med weboptimerede billedformater som PNG, JPG, SVG og WebP.
  • Webhosting: Opsætning og brug af personlige domæner og webhoteller til projekt-hosting.
  • Tilgængelighed og Etik: Overvejelser omkring bæredygtighed, inklusivitet og etiske principper i design.
  • Ophavsret: Forståelse for immaterielle rettigheder og metadata i digitale billeder.

Ærdigheder:

  • Udvikling af responsive og visuelt tiltalende websites.
  • Strukturering og styling af indhold med HTML og CSS
  • Skabelse af interaktive prototyper og brugervenlige designs i Figma.
  • Gennemførelse af brugertests og anvendelse af feedback til forbedringer.
  • Administration af webhosting og filoverførsler med FTP-værktøjer som FileZilla.

REFLEXION

Tema 2 var min allerførste erfaring med at udvikle et website, og det var en spændende og inspirerende oplevelse. Jeg var både udfordret og fascineret af processen, og det føltes utrolig tilfredsstillende at se mit arbejde tage form fra en idé til et færdigt produkt. Det var her, jeg for alvor begyndte at få en forståelse for, hvordan design og udvikling hænger sammen, og det motiverede mig til at dykke dybere ned i webudviklingens verden.

MOBILSITE

Projektet inkluderede:

  • Mobile-First Design: Optimeret til visning på forskellige enheder.
  • Visuel Hierarki: Stylet med CSS Grid og Flexbox for et rent og overskueligt layout.
  • Grundlæggende Funktioner: En enkel, funktionel side med fokus på struktur og responsivitet. design.
  • Domænehosting: Uploadet til et personligt domæne via et webhotel.

Teknologier Brugte:

HTML, CSS, Figma, FileZilla, browser developer tools.
